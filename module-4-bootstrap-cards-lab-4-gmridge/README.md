## Bootstrap Cards

Using only bootstrap elements, create this grid-based card layout, making it responsive.

<br/>

> ## Instructions

<br/>

1.) Update the index.html file with the Bootstrap application.

2.) Responsiveness should be as follows:

- 4 cards per row on large screens (and above),
- 2 cards per row on medium screens, and
- 1 card per row on small screens (and below).
