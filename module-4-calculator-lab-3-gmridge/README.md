# Module 4 | Lab 3

<br/>

## Calculator

Design and create a calculator. It should be practical and work well performing actions like +,-,_,/_

# Steps

1. Update the Requirements.md file for your calculator
2. Sketch your calculator on paper and upload an image of it to the images folder
3. Create high-fi design with figma and paste a link
4. Update the Design.md
5. Develop your calculator with HTML, CSS and Javascript

6. Push your code to github
